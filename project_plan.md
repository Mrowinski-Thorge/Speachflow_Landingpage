# SpeachFlow – Projekt Plan

## 1. Projektbeschreibung
SpeachFlow ist eine KI-gestützte App zum Üben von Präsentationen und Pitches. Nutzer können Vorträge aufnehmen und erhalten detailliertes KI-Feedback zu Sprache, Haltung, Mimik und Tempo. Zielgruppen: Schüler, Studenten, Business-Profis.

## 2. Seitenstruktur
- `/` – Landing Page (One-Pager mit allen Sektionen)

## 3. Core Features der Landing Page
- [x] Sticky Frosted Glass Header
- [x] Hero Section mit App-Mockup
- [x] Features Section (KI-Analyse, Sprache, Haltung, Mimik, etc.)
- [x] Live Test Section
- [x] Download Section (App Store / Play Store)
- [x] Testimonials Section
- [x] Pricing Section (Free, Schüler 2,99€, Plus 4,99€, Business 9,99€)
- [x] How it Works / FAQ Section
- [x] Footer

## 4. Design-System
- Hell-Modus: Warmes Off-White (#F7F6F2), Mint (#2BAE8E), Slate Dark (#2E3A4A)
- Dunkel-Modus: Warmes Dunkelgrau (#1A1A18), Mint (#2BAE8E)
- Frosted Glass, abgerundetes Design, Whitespace-fokussiert
- Animationen: Framer Motion / CSS Transitions

## 5. Backend / Third-party Integration
- Supabase: Nicht benötigt (reine Marketing-Website)
- Shopify: Nicht benötigt
- Stripe: Nicht benötigt (App-Store-Käufe)
- Formulare: Kontaktformular via get_form_url
